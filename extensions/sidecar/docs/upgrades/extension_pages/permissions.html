Permissions rationale

Store reviewers love this page. Users do too, occasionally.

Host permissions (e.g., https://chatgpt.com/*) Needed to inject the dock UI and attach to the composer area on ChatGPT pages.
Storage Used to persist user settings (dock layout, toggles, preferences).
Scripting / content scripts Used to mount UI into the page and respond to SPA navigation/layout updates.
Optional (declare only if you actually use them) Side panel, offscreen documents, clipboardWrite, etc. Only request permissions that are strictly required, and gate advanced features behind capability checks.
