**Name**
Sidecar (Core) – Dock for ChatGPT

**Short description (max ~132 chars)**
A lightweight dock + UI upgrades for ChatGPT: resizable input, wide mode, and code tools. Fun pack sold separately.

**Long description**
Sidecar (Core) enhances the ChatGPT web interface with a compact dock and practical UI upgrades.

Core includes:
- Dock toolbar with quick actions
- Resizable composer/input
- Wide mode layout toggle
- Code block utilities (collapse/copy improvements)
- Persistent settings

Optional Fun Pack (separate download):
- Desktop pets overlay
- Progress Quest idle game integration

**Important**
This extension adapts to ChatGPT UI updates and may require maintenance when the site changes.
