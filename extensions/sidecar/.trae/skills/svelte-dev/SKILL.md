# Svelte Development Skill

This skill provides comprehensive Svelte 5 development capabilities with TypeScript integration, focusing on modern Svelte features like runes, reactive programming, and Shadow DOM encapsulation.

## When to Use
- Creating standalone Svelte components
- Implementing Shadow DOM/CSS encapsulation
- Using Svelte 5 runes system
- TypeScript integration with Svelte
- Building self-contained UI components

## Key Features
- Svelte 5 with runes support
- TypeScript strict mode
- Shadow DOM encapsulation
- Custom event emission
- Props typing and validation
- Build configuration for vanilla JS output

## Development Guidelines
1. Use Svelte 5 syntax with $state, $derived, $effect runes
2. Implement proper TypeScript interfaces for props
3. Use ShadowRoot for complete CSS isolation
4. Emit custom events for parent communication
5. Follow Svelte best practices for component lifecycle
